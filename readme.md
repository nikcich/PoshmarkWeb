# Back End Directory
## Create virtual env
(Mac)
```python3 -m venv venv```
(Windows)
```python -m venv venv```

## Activate virtual env
(Mac)
```source venv/bin/activate```
(Windows)
```.\venv\Scripts\activate```
    
## Run python server
(Mac)
```python3 Server.py```
(Windows)
```python Server.py```

## Install flask
(Mac)
```pip3 install flask```
(Windows)
```pip install flask```

# Front End Directory-
## Start dev server at localhost:3000
```npm start```
    
## Create build for python to serve
```npm run build```
